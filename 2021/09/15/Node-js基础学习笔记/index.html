Node.js基础学习笔记

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。
Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。
Node.js 的包管理器 npm，是全球最大的开源库生态系统。

下载和安装

官方网站
官网下载，常规安装，不过多赘述

node -v //查看版本，验证安装

npm包管理器

npm是node官方的包管理器，用于下载第三方模块
可以通过npm管理一整个Node项目或者前端项目的所有依赖

npm -v //查看版本，验证安装

cnpm（国内环境的npm替代）

npm服务器在国外，访问慢
国内有cnpm，也就是npm的淘宝镜像

npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm -v //查看版本，验证安装

创建npm项目

在新建的npmprojcet文件夹下cmd，输入：

npm init

来进行初始化
会生成一个package.json文件，里面包含之前输入的项目信息

同步第三方依赖

cnpm instrall jquery --save   //--save的意思是将依赖集成到package.json里（差不多）

传输项目时只需要传输package.json文件（配置文件）
在另一台电脑上执行：

cnpm instrall

就会查找package.json文件，并将依赖下载到本地

模块化开发

外部模块

引入的外部模块有三种：

  1. 核心模块：node自带模块，可以使用require()引入后直接使用
  2. 自定义模块：自己编写的，需要指定路径
  3. 第三方模块：使用npm下载的第三方模块，下载完成后可以引用

自定义模块

新建两个.js文件:sum.js 和 main.js

  1. 在sum.js中书写模块方法：
    function sum(a, b) {
    return a + b;
}
module.exports = sum;//暴露模块接口
  2. 在main.js中引入外部模块：
    const sum = require('./sum');//引入一个外部模块
console.log(sum(10, 2));

核心模块

  1. 读取文件 readFile();函数

    const fs = require('fs');
fs.readFile('test.txt', (err, data) => {//文件路径、err未找到返回、data文件内容
    if (err) {
        console.log(err);
    }
    console.log(data.toString());

})

    readFile();函数有两个参数：

    • 读取文件的地址
    • 回调函数

  2. 处理文件路径 path.join(a,b,c);函数

    const path = require('path');
let a = 'http://www.baidu.com',
    b = 'doc',
    c = '666';
a = path.join(a, b, c);
console.log(a);

http模块

可以通过http模块的createServer方法在本地创建一个服务器：

const http = require('http');
const server = http.createServer((req, res) => {
    res.end('hello,world'); //响应内容,可通过浏览器输入127.0.0.1:端口查看
});
server.listen(3000, () => { //指定端口
    console.log('server is running'); //直接运行js的显示
})


此时服务器正在运行
在浏览器输入127.0.0.1:3000
显示hello，world
这样就完成了一个node服务器的搭建

之后每次更改都需要重启服务器
所以可以使用nodemon

cnpm install -g nodemon//下载
nodemon server.js//运行文件

修改文件后，服务器自动重启

在node服务器上渲染静态页面

const fs = require('fs');
const http = require('http');//调用

const server = http.createServer();//创建一个node服务器
server.listen(3000, () => {

});

server.on('request', function(request, res) {
    var url = request.url;

Koa框架

使用

npm init //初始化
cnpm install --save koa //安装Koa

使用koa创建node服务器

const Koa = require('koa'); //引入koa构造函数
const app = new Koa(); //创建应用
app.use(async(ctx) => { //中间件函数，在页面响应之前加载
    ctx.body = 'hello,koa';
})
app.listen(3000, () => { //监听端口
    console.log('服务器正在3000端口运行...');
})

路由

const Koa = require('koa'); //引入koa构造函数
const router = require('koa-router')() //引入并执行koa-router函数

const app = new Koa(); //创建应用

router.get('/', async(ctx) => {
    ctx.body = '<h1>index<h1>';
})
router.get('/about', async(ctx) => {
    ctx.body = 'about';
})
app.use(router.routes()); //将router作为中间件引入app

app.listen(3000, () => { //监听端口
    console.log('服务器正在3000端口运行...');
})
