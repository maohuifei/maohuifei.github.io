多重样式优先级深入概念

优先级是浏览器是通过判断哪些属性值与元素最相关以决定并应用到该元素上的。优先级仅由选择器组成的匹配规则决定的。
优先级就是分配给指定的CSS声明的一个权重，它由匹配的选择器中的每一种选择器类型的数值决定。

下列是一份优先级逐级增加的选择器列表：

  • 通用选择器（*)
  • 元素(类型)选择器
  • 类选择器
  • 属性选择器
  • 伪类
  • ID 选择器
  • 内联样式

!important 规则例外
当 !important 规则被应用在一个样式声明中时,该样式声明会覆盖CSS中任何其他的声明, 无论它处在声明列表中的哪里. 尽管如此, !important规则还是与优先级毫无关系.使用 !important 不是一个好习惯，因为它改变了你样式表本来的级联规则，从而使其难以调试。
一些经验法则：

  • Always 要优化考虑使用样式规则的优先级来解决问题而不是 !important
  • Only 只在需要覆盖全站或外部 css（例如引用的 ExtJs 或者 YUI ）的特定页面中使用!important
  • Never 永远不要在全站范围的 css 上使用!important
  • Never 永远不要在你的插件中使用!important
权重计算


解释：

  • 内联样式表的权值最高 1000
  • ID 选择器的权值为 100
  • Class 类选择器的权值为 10
  • HTML 标签选择器的权值为 1

CSS样式优先级

内联样式 > 内部样式表 > 外部样式表 > 浏览器样式表

css 7 种选择器：

  • id 选择器
  • 类选择器
  • 伪类选择器
  • 属性选择器
  • 伪元素选择器
  • 通配选择器
  • 标签选择器

css选择器优先规则

内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器

实例

HTML:

$ 
<div class="content-class" id="content-id"></div>

CSS:

$ 
#content-id {
    color: red;
}
.content-class {
    color: blue;
}
div {
    color: grey;
}

div的最终显示结果为红色

元素的显示和隐藏

display属性

display属性用于设置一个元素如何显示

  • display: none; 隐藏对象
  • display: block; 除了转换成块级元素外，还有显示元素的意思

diplay隐藏元素后不再占有原来位置
后面应用及其广泛，搭配js做网页特效

visibility 可见性

visibility 属性用于指定元素可见或者隐藏

  • visibility: visible; 元素可见
  • visibility: hidden; 元素隐藏

与 display 不一样的是，visibility 隐藏元素后继续占用原来位置
实际开发中，主要还是用 display

overflow 溢出

overflow 属性指定了如果内容溢出了一个元素的框（超过其指定的宽度和高度）时，会发生什么

  • overflow: visible; || 显示
  • overflow: hidden; || 隐藏
  • overflow: scroll; || 溢出部分显示滚动条
  • overflow: auto; || 在合适的时候显示滚动条，超出显示不超出不显示

精灵技术

使用精灵图核心：

  1. 精灵图主要针对于背景图片使用，就是把多个小背景图片整合到大图片中
  2. 这个大图片叫做 sprites 精灵图（雪碧图）
  3. 移动背景图片的位置 使用 background-position 属性
  4. 移动的距离是目标图片的 x 轴和 y 轴，注意网页中的坐标有所不同，一般数值都是负的（往上往左是负值）

字体图标（iconfont）

特点

  • 轻量级：文件小，和字体一起加载渲染，减少服务器请求
  • 灵活性：本质是文字，可以随意更改样式
  • 兼容性：几乎支持所有浏览器

样式结构比较简单，代替不了精灵图/图片

使用

下载

  • icomoon图标库：https://icomoon.io/
  • 阿里巴巴图标库：https://www.iconfont.cn/

引用

  1. 下载文件到本地后，解压压缩文件
  2. 将 fonts 文件夹放到 Web 根目录
  3. 将字体声明代码（解压后文件根目录的style.css开头）复制到html文件的 style 标签里
  4. 新建标签，内容为图标对应符号
  5. 给标签声明字体（可添加颜色字号属性）

追加

  1. 上传之前的图标记录文件（selection.json）
  2. 选择新的图标
  3. 打包下载到本地,替换原来的文件即可
