CSS传统网页布局之定位

CSS 学习笔记 Web前端
学习笔记
发布日期:   2021-08-03
更新日期:   2021-08-11
文章字数:   1.3k

在CSS中，通过CSS定位(CSS position)可以实现网页元素的精确定位。CSS定位和CSS浮动类似，也是控制网页布局的操作，CSS定位更加灵活，可以针对更多个性化的布局方案来使用。
定位分为静态定位，相对定位，绝对定位，固定定位这四种
本文参考：
菜鸟教程——CSS教程：https://www.runoob.com/css/css-tutorial.html
w3school——CSS教程：https://www.w3school.com.cn/css/index.asp
黑马程序员pink老师前端入门教程：https://www.bilibili.com/video/BV14J4114768

为什么需要定位

  • 浮动可以让多个块级盒子没有缝隙的排列显示，经常用于横向排列盒子
  • 定位可以让盒子自由的在某个盒子内移动位置或者固定在屏幕中的某个位置，并且可以盖住其他盒子

定位的组成

定位 = 定位模式 + 边偏移

边偏移

决定该元素的最终位置，有top、bottom、left、right 四个属性：

四种定位模式

静态定位（static）

静态定位是元素的默认定位方式，无定位的意思

语法

$ 
div {
  position: static;
}
  • 静态定位按照标准流特性摆放位置，没有边偏移
  • 在实际布局中很少用

相对定位（relative）

元素在移动位置的时候，是相对于它原来的位置来说的

语法

$ 
div {
  position: relative;
}

特点

  1. 相对于自己原来的位置来移动的（移动位置的参照点是他原来的位置）
  2. 原来在标准流的位置继续占有，后面的盒子当它仍然在（不脱标，继续保留原来位置）
    因此，相对定位没有脱标，主要用来限制绝对定位的

绝对定位（absolute）

元素在移动位置的时候，是相对于它的祖先元素来说的

语法

$ 
div {
  position: absolute;
}

特点

  1. 如果没有祖先元素或者祖先元素没有定位，则以浏览器为准定位（Document文档）
  2. 如果祖先元素有定位（相对、绝对、固定），则以最近一级的有定位的祖先元素为参考点移动位置
  3. 绝对定位不再占有原先位置（脱标）

“子绝父相”

如果子级使用绝对定位，父级需要用相对定位

  1. 子级盒子用绝对定位，不会占用原来的位置，可以放到父盒子的任何一个地方，不会影响其他兄弟盒子
  2. 由于级盒子绝对定位要求父元素也有定位，所以父盒子需要加定位限制子盒子在父盒子内显示
  3. 父盒子在布局时，需要占有位置，不然会影响后面的其他盒子。所以使用相对定位

总结：父级在布局中需要占有位置，所以用相对定位。子级不需要占有位置，使用绝对定位

固定定位（fixed）

将元素固定到浏览器可视区的某个位置，不会随着界面的滚动而动

语法

$ 
div {
  position: fixed;
}

特点

  1. 以浏览器的可视窗口为参考点移动元素
  2. 跟父元素没有关系
  3. 不随滚动条移动
  4. 不占有原先位置（脱标）

特殊：粘性定位（sticky）

可看成相对定位和固定定位的混合

语法

$ 
div {
  position: sticky;
  top：0px；         || 当元素距离浏览器可视区顶部距离为0时，变成粘性定位（固定）
}

特点

  1. 以浏览器的可视窗口为参考点移动元素
  2. 占有原先位置
  3. 必须添加 top、left、bottom、right 其中一个才有效
  4. 跟滚动搭配使用，兼容性较差，IE不支持

扩展

定位的叠放次序

语法

$ 
div {
  z-index: 1;  || z轴，没有单位
}

特点

  1. 取值范围范围正整数、负整数、0，默认是auto，数值越大，盒子越靠上
  2. 如果属性相同，则按照书写顺序，后来者居上
  3. 数值没有单位，不能加单位
  4. 标准流和浮动没有叠放次序，只有定位有

绝对定位的盒子的居中算法

加了绝对定位的盒子不能通过 margin:0 auto; 来水平居中，但是可以通过计算方法来实现水平垂直居中
水平居中

  1. left父盒子宽度的50%(负值)
  2. margin-left子盒子（自身）宽度的50%

垂直居中同理

定位的特殊特性

绝对定位和固定定位和浮动类似

  1. 行内元素添加绝对/固定定位，可以直接设置宽高
  2. 块元素添加绝对/固定定位，如果不指定宽高，默认大小是内容大小，不再占用一整行
  3. 脱标的盒子不会触发塌陷（外边距合并）

文本环绕

  1. 浮动的元素不会压住下面的标准流文字（浮动最开始的目的是做文字环绕效果）
  2. 绝对/固定定位的元素会压住下面的所有元素
