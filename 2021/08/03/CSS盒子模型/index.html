CSS盒子模型

CSS 学习笔记 Web前端
读书笔记与感悟
发布日期:   2021-08-03
更新日期:   2021-08-03
文章字数:   963

所有HTML元素可以看作盒子，在CSS中，”box model”这一术语是用来设计和布局时使用。
CSS盒模型本质上是一个盒子，封装周围的HTML元素，它包括：边距，边框，填充，和实际内容。
盒模型允许我们在其它元素和周围元素边框之间的空间放置元素。

盒子模型

不同部分的说明：

  • Margin(外边距) - 清除边框外的区域，外边距是透明的。
  • Border(边框) - 围绕在内边距和内容外的边框。
  • Padding(内边距) - 清除内容周围的区域，内边距是透明的。
  • Content(内容) - 盒子的内容，显示文本和图像。

内容content

没啥好说的，就是内容

边框 border

语法规则

border可以设置元素的边框，边框有三部分组成：边框宽度、边框样式、边框颜色

$ 
div {
  border-width:10px;   ||粗细10像素
  border-style:solid   ||solid实线/dashed虚线/dotted点线；
  border-color:#fff;   ||白色
}

简写

$ 
div {
  border:1px solid red; 没有先后顺序，只有通俗约定规范
}

进阶

分开定义样式
$ 
div {
  border-top: 1px solid red;
  border-bottom: 
  border-left: 1px dashed green;
}
合并相邻的边框
$ 
div {
  border-collapse:collapse;
}
边框会影响实际盒子大小

即我们所看到的区域大小=盒子大小+边框大小

  • 测量盒子大小时候不量边框
  • 如果测量时包含边框，后面需要减去width/height

内边距padding

语法规则

padding可以设置元素的内边距

简写

内边距也会影响盒子实际大小

如果盒子已有大小，添加内边距盒子会被撑大
解决方法参考边框

外边距margin

语法规则

margin可以设置元素的外边距，控制盒子和盒子间的距离

简写

margin简写意义顺序和pading完全一样

盒子模型的典型应用

外边距——块级盒子水平居中对齐

外边距可以让块级盒子居中，但必须满足两个条件：
1.块级盒子必须指定了宽度
2.盒子左右外边距都设置为auto

$ 
div {
  常见三种写法：
  margin:0 auto; ||推荐
  margin: auto;
  margin-left:auto;margin-right:auto;
}

行内元素和行内块元素的水平居中

给其父元素添加text-align:center即可

外边距合并，嵌套块元素的塌陷问题

对于两个嵌套关系（父子关系）的块元素，父元素有上外边距的同时子元素也有上外边距，此时父元素会塌陷较大的外边距值


其他方法：使用浮动、固定、绝对定位的盒子不会有塌陷问题

清除内外边距

$ 
* {
  padding:0;
  margin:0;
}

行内元素的兼容性问题

行内元素照顾兼容性问题，尽量只设置左右内外边距，不设置上下内外边距。转化为块级或者行内块级元素即可

圆角边框

基础语法

CSS3中，新增圆角边框样式

$ 
{
  border-radius:10px;  ||数值为四角圆的半径，可以是数值或者百分比
}

进阶

如果要生成圆形盒子，数值设为正方形盒子宽度的一半或者50%即可
常用圆角矩形数值为盒子高度的一半

简写

左上、右上、右下、左下（顺时针）

$ 
{
border-top-left-radius:10px;
border-top-right-radius:10px;
border-bottom-left-radius:10px;
border-bottom-right-radius:10px;
}

盒子阴影

CSS3中新增盒子阴影样式，使用 box-shadow 为盒子添加阴影

PS：默认的是外阴影outset，但是这个单词不允许出现，否则阴影无效
阴影不占用空间，不影响盒子布局排列

文字阴影

CSS3中新增文字阴影样式，使用 text-shadow 为文本添加阴影
