HTML5

什么是 HTML5?

  • HTML5 是下一代 HTML 标准。
  • HTML , HTML 4.01的上一个版本诞生于 1999 年。自从那以后，Web 世界已经经历了巨变。
  • HTML5 仍处于完善之中。然而，大部分现代浏览器已经具备了某些 HTML5 支持。

HTML5 中的一些有趣的新特性：

  • 用于绘画的 canvas 元素
  • 用于媒介回放的 video 和 audio 元素
  • 对本地离线存储的更好的支持
  • 新的特殊内容元素，比如 article、footer、header、nav、section
  • 新的表单控件，比如 calendar、date、time、email、url、search

新增多媒体标签

视频 video

<video src='文件地址' controls='controls'><video>

原生的支持视频文件的播放，但是支持的格式有限：

几乎所有浏览器都支持MP4格式。
为了适配不同浏览器用户，可以使用如下写法：

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
您的浏览器不支持Video标签。
</video>

<video> 元素支持多个 <source> 元素. <source> 元素可以链接不同的视频文件。浏览器将使用第一个可识别的格式

<video>元素也提供了 width 和 height 属性控制视频的尺寸:

  • 如果设置的高度和宽度，所需的视频空间会在页面加载时保留。
  • 如果没有设置这些属性，浏览器不知道大小的视频，浏览器就不能再加载时保留特定的空间，页面就会根据原始视频的大小而改变。

新增选择器

属性选择器” [] “

//选择有value属性的标签
input[value]{
  color: red;
}
//选择有type属性且值为text的标签
input[type=text]{
  color: blue;
}
//选择有class属性且值以con开头的div标签
div[class^=con]{
  color: red;
}
//选择有class属性且值以ico结尾的div标签
div[class$=ico]{
  color: red;
}
//选择有class属性且任意位置的值有ico的div标签
div[class*=ico]{
  color: red;
}

[]属性选择器的权重是10，权重等同于类选择器,伪类选择器

结构伪类选择器” : “

根据文档结构来选择,多用于选择父元素里面的某个子元素

//选择ul里面的第一个元素
ul:first-child{
  color: red;
}
//选择ul里面最后一个元素
ul:last-child{
  color: red;
}
//选择ul里面的第n个元素
ul:nth-child（n）{
  color: red;
}
  • n可以是数字，关键字，公式
  • 如果是数字，就是选择第n个元素，从1开始
  • 如果是关键字，就是even偶数，odd奇数
  • 如果是公式，括号内为n(只能是n)，就是从0开始，每次加1，超出部分忽略（相当于选择左右的子元素）


这个貌似挺重要的

后三个和前三个选择器用法基本一样

//选择ul里面的第一个元素
ul:first-of-type{
  color: red;
}
//选择ul里面最后一个元素
ul:last-of-type{
  color: red;
}
//选择ul里面的第n个元素
ul:nth-of-type（n）{
  color: red;
}

nth-child（n）和nth-of-type（n）的区别：

//使用nth-child（n）时：
<style>
  section div:nth-child（1){ //nth-child执行的时候，先判断nth-of-type(1)，再回头判断是否是div元素。所以没有满足条件的子元素，样式不生效
    color: red;
  }
</style>
<section>
  <p>光头强</p>
  <div>熊大</div>
  <div>熊二</div>
</section>
//最后结果是都不变，样式更改没生效

//使用nth-of-type（n）时：
section div:nth-of-type（1）{ //nth-of-type执行的时候，先选择div元素，再执行nth-of-type（1）。满足条件，结果是熊大变色
    color: red;
   } 。。。
//最后结果是熊大变色

伪元素选择器” :: “

利用CSS创建新标签元素，不需要HTML标签，简化HTML结构

  • before和after创建一个元素，但是属于行内元素
  • 新创建的元素在元素树中是找不到的，所以称之为“伪元素”
  • 语法：name::before{}
  • before和after必须有content属性（内容）
  • before在父元素前面创建元素，after在父元素后面插入元素
  • 权重等同标签选择器，为1

使用场景：

  1. 配合字体图标

  2. 鼠标经过，添加半透明黑色遮罩

  3. 伪元素清除浮动
    详情查看本站《CSS传统网页布局之浮动》——清除浮动

指定盒子模型大小 box-sizeing

  1. box-sizeing: content-box;
    盒子大小为width + border + padding（默认的）
  2. box-sizeing: border-box;
    盒子大小为width（就不用减去添加的样式宽度了）

一般直接放在CSS初始化样式里即可

滤镜filter

将模糊或颜色偏移等图形效果应用于元素

filter: 函数()   例如：filter: blur(5px); blur模糊处理，数值越大越模糊

常用属性：

  • 模糊 blur(px)
  • 亮度 brightness(%)
  • 对比度 contrast(%)
  • 透明 opacity(%)

弹性计算 calc函数

在声明CSS属性值时进行一些计算

width: calc(100%-30px);

括号里可使用+-*/来计算

过渡

当元素从一种样式变为另一种样式时为元素添加效果
从一个状态，渐渐过渡到另一个状态
经常和.hover搭配使用

语法

transition: 要过渡的属性 花费时间 运动曲线 何时开始；
  1. 属性：想要变化的css属性，宽高、背景颜色、内外边距都可以
    想要所有属性都变化，设置值为all即可
  2. 花费时间：必须写单位，单位是秒(s)
  3. 运动曲线：默认是ease
  4. 何时开始：必须写单位，单位是秒，可以设置延迟触发时间，默认是0


PS：一个transition属性可以有多个变化的属性，属性之间用逗号隔开
想要所有属性都变化，设置属性值为all即可
