HTML5

什么是 HTML5?

  • HTML5 是下一代 HTML 标准。
  • HTML , HTML 4.01的上一个版本诞生于 1999 年。自从那以后，Web 世界已经经历了巨变。
  • HTML5 仍处于完善之中。然而，大部分现代浏览器已经具备了某些 HTML5 支持。

HTML5 中的一些有趣的新特性：

  • 用于绘画的 canvas 元素
  • 用于媒介回放的 video 和 audio 元素
  • 对本地离线存储的更好的支持
  • 新的特殊内容元素，比如 article、footer、header、nav、section
  • 新的表单控件，比如 calendar、date、time、email、url、search

视频 video

<video src='文件地址' controls='controls'><video>

原生的支持视频文件的播放，但是支持的格式有限：

几乎所有浏览器都支持MP4格式。
为了适配不同浏览器用户，可以使用如下写法：

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
您的浏览器不支持Video标签。
</video>

<video> 元素支持多个 <source> 元素. <source> 元素可以链接不同的视频文件。浏览器将使用第一个可识别的格式

<video>元素也提供了 width 和 height 属性控制视频的尺寸:

  • 如果设置的高度和宽度，所需的视频空间会在页面加载时保留。
  • 如果没有设置这些属性，浏览器不知道大小的视频，浏览器就不能再加载时保留特定的空间，页面就会根据原始视频的大小而改变。

新增的3个选择器

属性选择器” [] “

//选择有value属性的标签
input[value]{
  color: red;
}
//选择有type属性且值为text的标签
input[type=text]{
  color: blue;
}
//选择有class属性且值以con开头的div标签
div[class^=con]{
  color: red;
}
//选择有class属性且值以ico结尾的div标签
div[class$=ico]{
  color: red;
}
//选择有class属性且任意位置的值有ico的div标签
div[class*=ico]{
  color: red;
}

[]属性选择器的权重是10，权重等同于类选择器,伪类选择器

结构伪类选择器” : “

根据文档结构来选择,多用于选择父元素里面的某个子元素

//选择ul里面的第一个元素
ul:first-child{
  color: red;
}
//选择ul里面最后一个元素
ul:last-child{
  color: red;
}
//选择ul里面的第n个元素
ul:nth-child（n）{
  color: red;
}
  • n可以是数字，关键字，公式
  • 如果是数字，就是选择第n个元素，从1开始
  • 如果是关键字，就是even偶数，odd奇数
  • 如果是公式，括号内为n(只能是n)，就是从0开始，每次加1，超出部分忽略（相当于选择左右的子元素）


这个貌似挺重要的

后三个和前三个选择器用法基本一样

//选择ul里面的第一个元素
ul:first-of-type{
  color: red;
}
//选择ul里面最后一个元素
ul:last-of-type{
  color: red;
}
//选择ul里面的第n个元素
ul:nth-of-type（n）{
  color: red;
}

nth-child（n）和nth-of-type（n）的区别：

//使用nth-child（n）时：
<style>
  section div:nth-child（1){ //nth-child执行的时候，先判断nth-of-type(1)，再回头判断是否是div元素。所以没有满足条件的子元素，样式不生效
    color: red;
  }
</style>
<section>
  <p>光头强</p>
  <div>熊大</div>
  <div>熊二</div>
</section>
//最后结果是都不变，样式更改没生效

//使用nth-of-type（n）时：
section div:nth-of-type（1）{ //nth-of-type执行的时候，先选择div元素，再执行nth-of-type（1）。满足条件，结果是熊大变色
    color: red;
   } 。。。
//最后结果是熊大变色

伪元素选择器” :: “

利用CSS创建新标签元素，不需要HTML标签，简化HTML结构

  • before和after创建一个元素，但是属于行内元素
  • 新创建的元素在元素树中是找不到的，所以称之为“伪元素”
  • 语法：name::before{}
  • before和after必须有content属性（内容）
  • before在父元素前面创建元素，after在父元素后面插入元素
  • 权重等同标签选择器，为1

使用场景：

  1. 配合字体图标

  2. 鼠标经过，添加半透明黑色遮罩

  3. 伪元素清除浮动
    详情查看本站《CSS传统网页布局之浮动》——清除浮动

指定盒子模型大小 box-sizeing

  1. box-sizeing: content-box;
    盒子大小为width + border + padding（默认的）
  2. box-sizeing: border-box;
    盒子大小为width（就不用减去添加的样式宽度了）

一般直接放在CSS初始化样式里即可

  • 模糊 blur(px)

过渡 transition

当元素从一种样式变为另一种样式时为元素添加效果
从一个状态，渐渐过渡到另一个状态
经常和.hover搭配使用

语法

transition: 要过渡的属性 花费时间 运动曲线 何时开始；
  1. 属性：想要变化的css属性，宽高、背景颜色、内外边距都可以
    想要所有属性都变化，设置值为all即可
  2. 花费时间：必须写单位，单位是秒(s)
  3. 运动曲线：默认是ease
  4. 何时开始：必须写单位，单位是秒，可以设置延迟触发时间，默认是0


PS：一个transition属性可以有多个变化的属性，属性之间用逗号隔开
想要所有属性都变化，设置属性值为all即可

2D转换 transform

Web的二维坐标系

translate: 移动

transform: translate(x,y);
//或者分开写
transform: translateX(n);
transform: translateY(n);

注意：

  1. 不会影响其他盒子的位置（浮起来，不脱标）
  2. 对行内标签没有效果
  3. x/y处填百分比是以 盒子本身的宽高 为基准，50%为移动盒子宽的一半的距离

利用这点可以更简单的实现盒子的水平垂直居中：

div {
  position: relative;
  w: 1000px;
  h: 1000px;
}
p {
  position: absolute;
  top: 50%;
  left: 50%;
  w: 200px;
  h: 200px;
  /*原来定位写法：
  margin-top: -100px;
  margin-left: -100px;
  冗余、且需要计算*/
  trabsform: translate(-50%,-50%);
}

rotate: 旋转

transform: rotate(45deg); //顺时针旋转45度，负值为逆时针，deg是单位必须写
transform-origin: x y;    //设置旋转中心点

注意：

  1. 默认中心点是 (50%，50%)
  2. x和y是用空格隔开的，不是逗号
  3. 值还可以是像素(px)或者方位名词(top、bottom、left、right、centen)

scale: 缩放

transform: scale(x,y);
 //x和y是倍数，放大一般是(2,2), 缩小一般是（0.5,0.5）
 //可以设置单边（2,1）一倍是不变
 //等比缩放简单写法：括号里只写一个倍数

和直接修改宽高相比：

  1. 缩放不会对其他盒子有影响
  2. 直接修改宽高它的变换中心点是上边缘中点，且没法改变

综合写法

属性间用空格隔开

transform: translate(x,y) rotate(x,y) scale(x,y)

同时有位移和其他属性的时候，位移放到最前面

动画 animathon

通过设置多个节点来精确控制一个或一组动画，常用来实现复杂的动画效果

基本使用

  1. 先定义动画
  2. 再使用（调用）动画

定义：

@keyframes 动画名称{
  0%: {
    width: 100px; background-coler: red; 
  }
  50% {
    width: 150px; background-coler: yellow;
  }
  100%: {
    width: 200px; background-coler: green;
  }
}

注意：

  1. 为了得到最佳的浏览器支持，应该始终定义 0% 和 100% 选择器
  2. 多个属性之间用分号隔开

调用：

 div {
      animation: 定义的名称（name） 动画的持续时间（s)，
      定义的名称（name） 动画的持续时间（s);
}

注意：
元素可以添加多个动画，动画之间用逗号隔开

所有动画属性

动画序列

  1. 0%是动画的开始，100是动画的结束。这样的规则就是动画序列
  2. 在 @keyframes 中规定某项CSS样式，就能创建由当前样式逐渐转变为新样式的效果
  3. 动画是使元素从一种样式逐渐变化为另一种样式的效果，可以改变任意多的样式任意多的次数。
  4. 请用百分比来规定变化发生的时间，或用关键词 “from” 和 “to”，等同于 0% 和 100%。

简写

div
{
    animation-name: myfirst;
    animation-duration: 5s;
    animation-timing-function: linear;
    animation-delay: 2s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
    animation-play-state: running;
}
................简写...................

div
{
    animation: myfirst 5s linear（匀速） 2s infinite（无限循环） alternate;
}

3D转换 transform

三维坐标系

  • X轴：水平向右，(右+/左-)
  • Y轴：垂直向下，(下+/上-)
  • Z轴：垂直于屏幕，(外+/里-)

3D位移 translate3d(x,y,z)

{
    transform: translateX(120px);
    transform: translateY(120px);
    transform: translateZ(120px);//直接使用无效果，需要借助下面的透视
    transform: translate3d(x,y,z);//简写
}

透视 perspective


d：视距，眼睛到屏幕的距离
z：Z轴，物体距离屏幕的距离

  • 透视也称为视距，人的眼睛到屏幕的距离
  • 近大远小： 距离视觉点越近的在电脑平面成像越大，越远成像越小
  • 透视单位是像素

透视写在 被观察元素 的 父盒子 上面

perspective和translateZ能实现相同的视觉效果
但是，透视是给父盒子加的属性，里面可能有多个子盒子

透视应用场景：

给父盒子一个perspective，每个子盒子通过translateZ更改Z轴前后关系

3D旋转 rotate3d(x,y,z)

让元素在三位平面内沿着X/Y/Z轴旋转

{
    transform: rotateX(120deg);//沿着X轴正方向旋转120度
    transform: rotateY(120deg);//沿着Y轴正方向旋转120度
    transform: rotateZ(120deg);//沿着Z轴正方向旋转120度

    transform: rotate3d(x,y,z,deg);
    //沿着自定义轴正方向旋转
    //沿X轴旋转180度：（1,0,0,180deg）
    //还可以沿着对角线进行旋转：（1,1,0,180deg）
}

左手守则

利用左手守则判断旋转方向：

  • 左手手拇指指向X轴正方向
  • 手指弯曲的方向就是该元素沿着X轴旋转的方向

3D呈现 transform-style

  1. 控制子元素是否开启三维立体环境

  2. transform-style: flat 默认是子元素不开启3D环境

  3. transform-style: preserve-3d 子元素开启立体环境

  4. 这个属性给父元素加，影响的是子盒子

  5. 重要属性

  6. 自己试验发现这个属性给上一级的父盒子添加才有用
