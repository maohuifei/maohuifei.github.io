Hexo matery 主题的个性化配置

Hexo 博客 主题 matery
Hexo博客主题配置
发布日期:   2021-04-16
更新日期:   2021-05-26
文章字数:   688

个人收录整理的 Hexo matery 主题 官方教程之外的 进阶实用个性化配置(不定时更新)

顶部栏颜色的更改

打开 matery/source/css/matery.css 文件 搜索 .bg-color
修改background-image属性中linear-gradient内的颜色色值（具体色值可以百度或者用PS查看）
色值相同是单色效果，两个色值不同就是渐变效果

顶部Banner的图片和文章特色图的更改

在 /source/medias/banner 文件夹中，把喜欢的图片重命名覆盖原来的图片即可，主题代码中默认是每天切换
在图片网站中找到喜欢的图片，右击复制图片链接。粘贴图片链接到新建文章的头文件img参数后面即可

顶部Banner颜色滤镜的删除

打开 matery/source/css/matery.css 文件 搜索 .bg-color

注释掉图中灰色部分

字体的更改

在博客根目录下的 source 文件夹内创建一个名字 font 的文件夹，放入字体ttf文件。
找到 /themes/matery/source/css/my.css 文件，填入下面的代码：

$ @font-face{
    font-family: 'myFont';
    src: url('../font/myFont.ttf');
}

.diyfont{
    font-family: 'myFont';
}

这里是创建了一个 diyFont 的类
将上面的 myFont 改成你自己的字体名称
在网站中右击检索，找到对应字体所使用的样式关键词

局部样式更改在 matery/source/css/matery.css 中 ctrl+F 搜索关键词
标题样式更改在 /themes/matery/layout/_partial/bg-cover-content.ejs 中

$ <div class="title center-align diyfont">
     <% if (config.subtitle && config.subtitle.length > 0) { %>
         <%= config.subtitle %>
      <% } else { %>
             subtitle
      <% } %>
 </div>

在样式代码的 class=“” 中更改之前定义的样式名字 diy-font
打包上传测试

Banner上的“开始阅读”按钮和下方图标的颜色更改

在网站中右击检索，找到对应位置所使用的样式关键词
局部样式更改在 matery/source/css/matery.css 中 ctrl+F 搜索关键词

更改新建文章时自动生成的各项头文件参数

在博客根目录下，找到 scaffolds 文件夹中的 post.md
更改模板内容，以后每次新建文章时就会自动生成添加的参数

在你的博客文章末尾添加版权声明

打开主题配置文件 /themes/next/_config.yml 。Ctrl+F搜索，找到 creative_commons，将 post 参数改为true。

更改首页Benner屏幕的占比

在 /css/my.css 中添加代码

$ 
@media only screen and (min-width: 993px) {
  #articleContent img {
    zoom: 50%;
    max-width: 80%;
    min-width: 60%;
  }

  .index-cover{
    max-height: 70vh;
  }

  .index-card {
    margin-top: -50px;
  }
}

index-cover 的值为比列
